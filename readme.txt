== Descripción ==
Mejora el control de ventas y productos en woocommerce a través de un dashboard de con gráficos

== Instalación ==
1. Sube la carpeta del plugin al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú 'Plugins' en WordPress.
3. (Opcional) Configura los ajustes desde el panel de administración.

== Preguntas frecuentes ==
= ¿Este plugin es compatible con multisite? =
Sí, funciona perfectamente en instalaciones multisite.

= ¿Hay versión premium? =
No, este plugin es completamente gratuito.
